# База данных

Проект реализует локальную базу данных на компьютере, позволяющую хранить различные таблицы и проводить все CRUD-операции над данными в них.
Данные в одном столбце имеют единый тип. Поддерживаемые типы данных: `int`, `str`, `bool`.

## Требования к установке

* **Python** от 3.13;
* **Poetry** от 2.0.0 до 3.0.0;
* **Ruff** от 0.14.6;
* **Prompt** от 0.4.1 до 0.5.0
* **Prettytable** от 3.17.0 до 4.0.0

и/или

* **GNU Make**.

## Установка

Распакуйте проект в удобную директорию. После выполнения требований введите в установленной директории:
```
make install
```
или
```
poetry install
```
После успешной установки можно начать пользоваться:
```
make run
```
или
```
poetry run database
```

## Управление таблицами

```
Список основных команд:

> create_table <имя_таблицы> <столбец1:тип> <столбец2:тип> .. - создать таблицу.
  Поддерживаемые типы данных: int, str, bool.
> list_tables - показать список всех таблиц.
> drop_table <имя_таблицы> - удалить таблицу. После корректного ввода команды впишите 'y', 'yes', 'да' для потдверждения команды или любое другое слово для отмены.
> exit - выход из программы.
> help - справочная информация.
```

## CRUD-операции

```
Операции с данными:

> insert into <имя_таблицы> values (<значение1>, <значение2>, ...) - создать запись.
> select from <имя_таблицы> where <столбец> = <значение> - прочитать записи по условию.
> select from <имя_таблицы> - прочитать все записи.
> update <имя_таблицы> set <столбец1> = <новое_значение1> where <столбец_условия> = <значение_условия> - обновить запись.
> delete from <имя_таблицы> where <столбец> = <значение> - удалить запись. После корректного ввода команды впишите 'y', 'yes', 'да' для потдверждения команды или любое другое слово для отмены.
> info <имя_таблицы> - вывести информацию о таблице.
```
